# FlightController  
A simple flight controller from RabbitLab.  
Main HARDWARE:  
  *SMT32F103ZETb  
  *BlueTooth module:LY-BL001  
  *MPU9250 for Accel,GYRO and MAG Sensor  
  *FS-iA10B for 2.4GHz 10 CHANNELS remote controller data receiver
